# News Parser (Python)

Simple web parser that collects the latest news headlines from a website.

## Features
- Parses website pages
- Extracts headlines
- Displays top 10 results
- Easy to modify for any website

## Technologies
- Python
- Requests
- BeautifulSoup

## Installation

1. Clone repository:
git clone https://github.com/zit-w/web_parser.git

2. Install dependencies:
pip install -r requirements.txt

3. Run script:
python parser.py

## Use cases
- Data collection
- Website monitoring
- Content analysis
- Automation tasks
